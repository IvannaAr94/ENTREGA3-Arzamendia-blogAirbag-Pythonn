# Entrega 3 - Tu Primera Página (Django)

Proyecto web desarrollado en Django para la tercera entrega del curso, utilizando el patrón MVT (Modelo - Vista - Template).

## 📌 Tema del proyecto

Blog musical dedicado a la banda Argentina de rock **Airbag**, donde se pueden gestionar publicaciones, integrantes y álbumes.

---

## ✅ Requisitos cumplidos

- ✔ Proyecto Web Django con patrón MVT.
- ✔ Herencia de plantillas HTML mediante `base.html`.
- ✔ Tres modelos en `models.py`:
  - `Post`
  - `Integrante`
  - `Album`
- ✔ Un formulario para cargar datos en cada modelo.
- ✔ Un formulario de búsqueda sobre publicaciones.
- ✔ Funcionalidad para eliminar publicaciones desde la web.
- ✔ README con instrucciones de uso y pruebas.
